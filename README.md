# 🚀 Portfólio Pessoal – Angular 19 + Docker + Nginx

Este é o código-fonte do meu **portfólio profissional**, desenvolvido com **Angular 19**, projetado para apresentar minhas experiências, habilidades e projetos de forma moderna, rápida e responsiva.

Além do desenvolvimento tradicional em Angular, o projeto também conta com suporte completo para **build de produção via Docker**, utilizando uma imagem leve do **Nginx** para servir os arquivos finais.

---

## 🎯 Objetivo do Projeto

A intenção deste portfólio é:

- Demonstrar minhas habilidades técnicas como desenvolvedor.
- Centralizar experiências profissionais, habilidades e projetos.
- Criar uma interface moderna e elegante, com navegação fluida.
- Facilitar o deploy em qualquer servidor utilizando **Docker**.

O projeto é totalmente responsivo e segue boas práticas de estruturação Angular.

---

## 🛠️ Tecnologias Utilizadas

### **Frontend**
- Angular 19
- Angular Material
- Bootstrap 5 + Icons
- FontAwesome
- NgBootstrap
- SCSS

### **Build & Infraestrutura**
- Node.js 22.x
- npm 10.x
- Docker (multi-stage build)
- Nginx para deploy de produção
